# Chem
## 项目简介：
本项目是关于化学分子，用于化学分子结构的编辑、保存和展示。
## 功能介绍：
## 核心技术选型：
 - TypeScript
 - React
## 工程化架构：
 - monorepo + pnpm workspaces
## 目录架构：
 - packages
   - ChemCore 核心库
   - Chem2DView 化学分子编辑器
   - Chem3DView 化学3D结构展示
   - ChemUI 化学UI组件库
   - ChemCharts 化学图表组件库
   - ChemTemplate 模板
 - test 单元测试
 - docs 文档
